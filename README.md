# SRathinaGiri.github.io

Personal project portfolio for [Rathina Giri](https://github.com/SRathinaGiri).

The site loads public, non-fork, non-archived repositories from the GitHub API and displays
them as searchable, categorized project cards.

## Customize

Edit `config.js` to:

- Feature selected repositories.
- Hide repositories.
- Improve a repository description.
- Override its title, categories, or live-demo URL.

Repository topics, descriptions, homepage URLs, and GitHub Pages settings should be maintained
on GitHub. The portfolio will pick up those changes automatically.

## Publish

Create the public repository `SRathinaGiri/SRathinaGiri.github.io`, push the `main` branch, and
configure GitHub Pages to deploy from the root of `main`.
